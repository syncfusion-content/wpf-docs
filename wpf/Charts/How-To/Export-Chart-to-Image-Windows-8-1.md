# Export Chart to Image (Windows 8.1) in WPF Charts

The export chart to image feature in the SfChart control enables the user to export the image of the chart in different image file formats. 

## Supported Formats

* JPG or JPEG
* JPG-XR
* GIF
* TIFF
* PNG
* BMP
